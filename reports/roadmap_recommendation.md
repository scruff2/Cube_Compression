# Roadmap Recommendation (Current State)

## Gate Outcomes

- Phase 0: pass
- Phase 1: pass_with_limitations (no larger real narrow-domain corpus yet)
- Phase 2 Gate A: fail (descriptor estimate-vs-actual gap still large)
- Phase 3 Gate B: fail (wins not repeatable enough)

## Decision

Current evidence supports a **pivot/stop** recommendation for cube-first investment.

## Why

1. Real cube modes still generally lose to the family-aware baseline on key preset runs.
2. Descriptor-coding refinements improved overhead but did not close the gap enough.
3. Scaling introduced larger capacity and longer route spans, but did not produce robust repeatable wins under the present evaluation setup.

## Recommended Next Step

- Prioritize family-aware structured coding as the practical path.
- Keep cube codec as a research branch only if new real-corpus evidence appears.

## Re-open Criteria

Re-open cube investment only if one of these is met:

- A larger real narrow-domain corpus shows repeatable real-mode cube wins vs family-aware.
- Descriptor estimate-vs-actual gap is reduced to near parity across multiple regimes.
